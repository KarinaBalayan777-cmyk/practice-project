## 📘 Описание проекта

Данный репозиторий содержит выполненные кейс-задачи в рамках производственной (технологической) практики по направлению веб-разработки.

Проект включает разработку учебных веб-приложений с использованием технологий:

- HTML
- CSS
- JavaScript (Vanilla JS)

## 📂 Структура репозитория

---

## 🧩 Кейс-задачи

### ✅ Кейс-задача №1
Анализ организации ООО «Гепард»:
- название и форма компании;
- виды деятельности;
- территориальное размещение;
- внешние факторы.

---

### ✅ Кейс-задача №2
Анализ структуры компании:
- организационная структура;
- функции сотрудников;
- преимущества и недостатки;
- уровень цифровизации.

---

### ✅ Кейс-задача №3 — Блог-приложение
Веб-приложение с функционалом:

- регистрация и вход пользователя;
- создание и редактирование постов;
- подписки на пользователей;
- комментарии;
- теги и сортировка;
- публичные и приватные посты.

Технологии:
- HTML / CSS / JavaScript

---

### ✅ Кейс-задача №4 — Веб-магазин книг
Функционал:

- интерфейс пользователя и администратора;
- просмотр книг;
- фильтрация (автор, категория, год);
- аренда книг;
- управление книгами.

---

### ✅ Кейс-задача №5 — Дневник путешествий
Функционал:

- добавление путешествий;
- просмотр записей других пользователей;
- геолокация;
- изображения;
- дополнительные параметры поездки.

---

## ⚙️ Используемые технологии

- HTML5
- CSS3
- JavaScript (ES6)
- DOM manipulation
- Event handling

## 📊 Цель проекта

Закрепление практических навыков веб-разработки, анализа систем и создания интерактивных веб-приложений.
